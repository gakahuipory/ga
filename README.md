# 圖途論戰 Graph Rivals

這是一款結合 **一筆畫** 與 **中國郵遞員問題** 的四人即時對戰遊戲。  
四位玩家輪流佔領與自己區域相連的邊，每條邊佔領時會擲骰決定分數，最終統計各玩家佔領的邊與總分。  
遊戲支援弱勢方申請機制，讓遊戲更具策略性與趣味性。

遊戲使用 Firebase Realtime Database 實現即時連線，支援房間機制、等待大廳、自動清理閒置房間。

## 🎮 核心玩法

1. **設定起點**  
   每位玩家在遊戲開始前，從 A～M 中選擇一個節點作為自己的起點。起點會以虛線圓圈標示（藍色為玩家1，紅色為玩家2，綠色為玩家3，橙色為玩家4）。

2. **佔邊規則**  
   - 輪到自己時，只能選擇**與自己現有區域相連的空邊**（起點或已佔邊的兩個端點視為自己的區域）。  
   - 點擊下方按鈕即可佔領該邊，不可佔領與自己區域不相連的邊。  
   - 每條邊只能被一位玩家佔領。

3. **骰子分數**  
   佔領邊的同時，系統會自動擲兩顆骰子（弱勢申請時為三顆，被佔方額外回合為一顆），兩骰點數和即為該邊的分數。分數會記錄在資料庫中，並顯示在按鈕上（例如 `AB(7)`）。

4. **弱勢方申請**  
   - 當某位玩家在自己的回合沒有任何可佔的邊時，會自動進入「弱勢申請階段」。  
   - 弱勢方可向與自己相鄰的強勢方（其他玩家）申請佔領一條邊。  
   - 申請成功後，弱勢方擲三顆骰子決定該邊分數，被佔方獲得一次額外回合（只需擲一顆骰子）。  
   - 申請的邊必須是強勢方擁有且與弱勢方相鄰，且不會導致強勢方圖形斷開（目前版本僅檢查相鄰）。

5. **遊戲結束**  
   當所有 28 條邊都被佔滿時，遊戲結束。每位玩家會進入結算頁面，觀看自己佔領的邊所形成的圖形，以及每條邊的分數。

## 🕹️ 遊戲流程

1. **建立／加入房間**  
   - 輸入房間號碼（3-7位數字），點擊「建立房間」成為房主（玩家1）。  
   - 其他玩家輸入相同房間號碼，點擊「加入房間」依序成為玩家2、3、4。

2. **等待大廳**  
   - 房主可以看到「開始遊戲」按鈕，但必須等待四位玩家全部加入後才能點擊。  
   - 玩家列表會即時顯示誰已加入（✅ 表示已加入）。

3. **起點選擇**  
   - 房主點擊「開始遊戲」後，所有玩家進入起點選擇階段。  
   - 每位玩家點擊下方 A～M 按鈕選擇自己的起點（只能選一次）。  
   - 當四位玩家都選完起點後，遊戲自動進入「進行中」階段。

4. **進行遊戲**  
   - 回合順序固定：玩家1 → 玩家2 → 玩家3 → 玩家4 → 玩家1 ...  
   - 輪到自己時，下方會出現可佔領的邊按鈕（僅顯示與自己區域相連的空邊）。  
   - 點擊按鈕即可佔領，同時擲骰並切換給下一位玩家。  
   - 若無邊可佔，自動進入弱勢申請階段，出現可申請的邊按鈕。

5. **遊戲結束**  
   - 所有邊佔滿後，自動跳轉至結算頁面，顯示每位玩家的總分與邊數，並繪製該玩家的佔邊圖形（邊上標記分數）。  
   - 點擊「返回大廳」可回到登入頁面。
