# 圖途論戰 Graph Rivals

這是一款結合 **一筆畫** 與 **中國郵遞員問題** 的四人即時對戰遊戲，靈感源自《遊戲人生》的世界觀。  
玩家從起點出發，輪流佔領地圖上的邊（靈脈），佔領時擲骰（或手動輸入）決定分數，最終以佔領的邊數與總分決勝。  
遊戲支援 **一般模式**（四人快節奏對戰）與 **派對模式**（一名裁判主持，其餘玩家手動輸入骰子點數），並提供弱勢方逆襲機制與裁判「返回上一步」功能。

## 🎮 核心玩法

### 1. 起始設定
- 每位玩家選擇一個節點作為起點（虛線圓圈標示）。
- 一般模式：藍、紅、綠、紫四隊（玩家1~4固定對應）。
- 派對模式：一名玩家擔任裁判（玩家1，不參與遊戲），其餘四名玩家（玩家2~5）固定為藍、紅、綠、紫隊。

### 2. 佔邊規則
- 輪到自己時，只能選擇 **與自己現有區域相連的空邊**（起點或已佔邊的端點視為區域）。
- 點擊下方按鈕佔領該邊，並根據模式擲骰：
  - 一般模式：自動擲兩顆骰子（弱勢申請時三顆，額外回合一顆）。
  - 派對模式：手動輸入骰子點數（玩家在真實世界擲骰後輸入），裁判可透過「返回上一步」糾正錯誤。
  - **輸入限制**：派對模式下只能輸入純整數，不可包含小數、符號或運算式（如 `2+2`）。
- 每條邊只能被一位玩家佔領，分數顯示在邊按鈕上（例如 `AB (7)`）。

### 3. 弱勢方逆襲機制（盟約宣誓）
- 當某位玩家在自己的回合沒有任何可佔的邊時，自動進入「弱勢申請階段」。
- 弱勢方可向 **與自己相鄰的強勢方邊** 提出申請，但該邊不能是強勢方圖形中的「橋」（移除後會使對方圖形分裂成兩個有邊的部分）。
- 申請成功後：
  - 弱勢方擲三顆骰子（派對模式手動輸入 3-18），獲得該邊分數。
  - 被佔方獲得一次額外回合（僅擲一顆骰子）。
  - 回合順序恢復正常（被佔方回合後，回到弱勢方原本的下一家）。

### 4. 遊戲結束
- 所有邊被佔滿後，進入結算頁面。
- 每位玩家（裁判除外）看到自己佔領的邊形成的圖形，並顯示總分。
- 派對模式下，裁判可觀看所有邊的全局圖形（各隊顏色標示），並查看邊成本列表。

## 🕹️ 遊戲流程

### 等待房間
- 房主（玩家1）輸入房間號碼建立房間，其他玩家輸入相同號碼加入。
- 房主可切換「一般模式」或「派對模式」（僅在無其他玩家時）。
- 四人全數加入後，房主點擊「開始遊戲」。
- 派對模式中，裁判會先選擇地圖（標準版或加大版），之後才進入起點選擇。

### 起點選擇
- 每位玩家點擊地圖上的節點按鈕選擇自己的起點（只能選一次）。
- 當四人全選完，遊戲自動進入「進行中」階段。

### 進行遊戲
- 回合順序固定：玩家1→2→3→4（一般模式）或玩家2→3→4→5（派對模式）。
- 輪到自己時，下方會出現可佔領的邊按鈕（只顯示與自己區域相連的空邊）。
- 點擊按鈕後，根據模式擲骰（一般模式自動，派對模式彈出輸入框）。
- 佔領後，該邊會以隊伍顏色顯示在地圖上，分數顯示在按鈕上。
- 若無邊可佔，自動進入弱勢申請階段，出現可申請的邊按鈕（只顯示不會使強勢方圖形分裂的邊）。

### 裁判專用功能（派對模式）
- 裁判頁面隱藏技能卡，改為顯示「返回上一步」按鈕。
- 畫面右上角「👨‍⚖️ 裁判模式」右側會顯示一個彩色圓圈，內含當前回合隊伍的第一個字（藍、紅、綠、紫），圓圈顏色與隊伍一致，方便裁判即時掌握回合狀態。
- 點擊「返回上一步」可回滾到上一步行動，讓該回合玩家重新選擇邊和輸入分數（歷史記錄自動保存，支援多步返回）。
