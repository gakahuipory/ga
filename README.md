# 一筆畫·中國郵遞員競技（雙人聯機）

這是一個結合 **一筆畫** 與 **中國郵遞員問題** 的雙人棋盤對戰遊戲。  
兩位玩家輪流佔領地圖上的邊，最終根據各自佔領的邊形成圖形。  
遊戲使用 Firebase Realtime Database 實現即時聯機，支援房間機制與技能卡系統。

---

## 🗺️ 地圖結構
節點：A, B, C, D, E, F  
邊連接（共 10 條）：
- A — B, C, D
- B — A, C, D
- C — A, B, D, F
- D — A, B, C, E, F
- E — D, F
- F — C, D, E

邊 ID 列表：AB, AC, AD, BC, BD, CD, CF, DE, DF, EF

---

## 🎮 核心玩法
1. **設定起點**：兩位玩家分別從 A~F 中選擇一個自己的起點。
2. **輪流佔邊**：從玩家 1 開始，每次點擊兩個相鄰節點佔領一條邊。
   - 必須至少有一個節點屬於自己當前已佔領的點集（起點或已佔邊的端點）。
   - 佔領後，該邊的兩個端點都加入自己的點集。
   - 每條邊只能被一方佔領。
3. **遊戲結束**：所有邊被佔滿後，顯示雙方佔領的邊構成的圖形。

---

## 🎲 骰子機制
- 每回合自動擲三顆骰子，範圍 3~18，顯示總和。
- 骰子結果可作為趣味隨機元素，或影響技能使用（如重骰）。

---

## 🔮 技能卡系統
- 每局開始時，每位玩家獲得 **2 張局內技能卡**（換線、重骰各 1 次）。
- **技能一（換線）**：與對手交換一條邊的歸屬。交換後雙方重新擲骰（回合切換）。
- **技能二（重骰）**：擲骰後，可以選擇 1~3 顆骰子重擲一次。
- **特殊卡**：可從「遊戲一」繼承（本版本提供模擬按鈕），使用後可增加任意技能使用次數 1 次。

---

## 🌐 聯機方式
- 使用 Firebase Realtime Database 同步遊戲狀態。
- 房主建立房間，對手輸入相同房間號加入。
- 支援雙人在不同設備上即時對戰。
